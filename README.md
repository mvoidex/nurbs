# nurbs

[![Hackage version](https://img.shields.io/hackage/v/nurbs.svg?style=flat)](http://hackage.haskell.org/package/nurbs) [![Build Status](https://travis-ci.org/mvoidex/nurbs.png)](https://travis-ci.org/mvoidex/nurbs)
